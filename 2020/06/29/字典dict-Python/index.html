字典dict-Python

映射

映射是一种键(索引)和值(数据)的对应

由用户确定，非默认

字典类型是“映射”的体现

键值对：键是数据索引的扩展

字典是键值对的集合，键值对之间无序

采用大括号{}和dict()创建，键值对用冒号: 表示

空字典 {}

查找时字典效率高

dict

使用dict从映射或键-值对创建字典

函数或方法 描述
del d[k] 删除字典d中键k对应的数据值
len(d) 返回字典d中元素的个数
d[k] 返回k所对应的值
d[k] = v 将值v关联到k
k in d 判断键k是否在字典d中，如果在返回True，否则False
d.keys() 返回字典d中所有的键信息
d.values() 返回字典d中所有的值信息
d.items() 返回字典d中所有的键-值对信息
d.update(c) 将c字典的键-值对添加到d中，若相同键有不同值，就用c的替换d的
d.get(k, ) 键k存在，则返回相应值，不在则返回值，默认是None
d.setdefault(k, ) 键k存在，则返回相应值，不在则在d中加入此键-值对
d.pop(k, ) 键k存在，则取出相应值，并将该键-值对删除，不在则返回值
d.popitem() 随机从字典d中取出一个键值对，以元组形式返回，并将该键-值对删除
d.clear() 删除所有的键值对
d.copy 返回一个新的字典，浅复制，副本与原件的值的指向相同。当替换副本中的值，正常。当修改副本中的值原件中的值也会改变。
d.fromkeys([], None) 创建字典，包含指定的键。值默认是None，也可以是特定的。
